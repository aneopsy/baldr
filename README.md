## Getting Started

If you want to run the project locally you can follow these steps.

### Prerequisites

Baldr use yarn but you can use npm instead.

- yarn

```sh
npm install -g yarn
```

### Installation

1. Clone the repo

```sh
git clone https://github.com/aneopsy/baldr
```

3. Install NPM packages

```sh
yarn install
```

4. Start baldr

```sh
yarn start
```
